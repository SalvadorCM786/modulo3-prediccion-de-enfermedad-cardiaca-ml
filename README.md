# modulo3-prediccion-de-enfermedad-cardiaca-ml

Proyecto de Machine Learning para predecir la presencia de enfermedad cardíaca utilizando Python, Google Colab y algoritmos de clasificación.

## Objetivo

Construir un modelo de aprendizaje automático supervisado capaz de clasificar si una persona presenta o no riesgo de enfermedad cardíaca, con base en variables clínicas del dataset.

## Herramientas utilizadas

- Python
- Google Colab
- Pandas
- NumPy
- Matplotlib
- Scikit-learn
- GitHub

## Dataset

El dataset utilizado corresponde a información clínica relacionada con enfermedades cardíacas.  
Incluye variables como edad, sexo, presión arterial, colesterol, frecuencia cardíaca máxima y otros indicadores médicos.

## Modelo utilizado

Se probarán modelos de clasificación como:

- Regresión logística
- Random Forest
- K-Nearest Neighbors
- Árbol de decisión

## Evaluación

El rendimiento del modelo se medirá usando:

- Accuracy / Exactitud
- Matriz de confusión
- Precision
- Recall
- F1-score
